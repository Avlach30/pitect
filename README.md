## Tech Stack
* `NestJS`
* `MySQL`
## Setup
1. Download/clone this repository. Or if you have an existing this repository on your local machine, you can pull/fetch this repository
2. Import file [`pitect.sql`](pitect.sql) in mysql database 
3. Run command `npm install` for installing all packages
4. Run command `npm run start:dev` for running application in local
